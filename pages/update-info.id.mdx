<Callout type="info">
- **Avatar**: Gunakan foto berukuran 1:1 atau persegi. Atau gunakan tautan dari foto yang kamu unggah di [imgbb.com](https://imgbb.com).

</Callout>

### Tampilkan/Sembunyikan Sosial Media
- Cari `type: social` dan ubah `on:true|false`.

| Sosial Media           | on (Bawaan)    | 
| :------------ | ----------: |
| `Truth Social`       | `true`          | 
| `Twitter`       | `true`          | 
| `Instagram`     | `false`          | 
| `Medium`        | `false`          | 
| `Github`        | `true`          | 
| `Youtube`       | `true`          | 
| `Linkedin`      | `true`          | 
| `Email`         | `true`          |
| `Tiktok`    | `false`          | 
| `Post.news`      | `false`          | 

<Callout type="info">
- `on: true` untuk menampilkan dan `on: false` untuk menyembunyikan. 
- Ikon sosial media berada di bawah identitas dan deskripsi. Jika kamu ingin menggunakan tampilan daftar, ubah menjadi `type: 'other'`.
</Callout>

### Tambah Tautan Sosial Media Baru

- Buat blok baru dengan menyalin perintah ini:
```js filename="data/LinksData.js" copy
{
   title: 'Social Name',
   url: 'https://example.com/',
   type: 'social',
   icon: '/icon.png',
   on: true
}
```

<Callout type="info">
Ubah semua info dan pastikan kamu menambahkan `icon.png` di folder [public](../directory-structure).
</Callout>
