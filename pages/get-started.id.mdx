import { Callout, FileTree } from 'nextra-theme-docs'

- Clone/Fork [repo ini](https://github.com/agcrisbp/ADLink) atau jalankan perintah berikut untuk membuat proyek baru:

```bash copy
yarn create next-app app -e https://github.com/agcrisbp/ADLink &&
cd app &&
yarn dev
```

- Buka [localhost:3000](http://localhost:3000) di browser kamu untuk melihat hasilnya.
<Callout type="info">
Gunakan editor apapun untuk mengedit. Baca [Aplikasi Edit](../app-to-use).
</Callout>

---

### Pratinjau

<Callout type="warning">
Untuk menghindari kegagalan akibat dari kesalahan kode ataupun penulisan, disarankan untuk menjalankan perintah ini terlebih dahulu sebelum meluncurkan aplikasi ke situs web kamu.
</Callout>

- Build hasil development yang sudah kamu edit.

```bash copy
yarn build
```

- Jika berhasil, maka aplikasi sudah siap diluncurkan.

- Atau jalankan dengan menggunakan perintah:

```bash copy
yarn start
```
