---
title: Struktur Direktori
---
Ini adalah beberapa struktur direktori untuk mengedit dan menjalankan ADLink.

```bash filename="ADLink"
┣ 📂 components/
┃ ┗ 📁 icons
┃ ┗ 📄 Layout.js
┃ ┗ 📄 Online.tsx
┃ ┗ 📄 ReusableStyles.js
┃ ┗ 📄 Seo.js
┃ ┗ 📄 WebLinks.js
┣ 📂 data/
┃ ┗ 📄 BioData.js
┃ ┗ 📄 LinksData.js
┣ 📁 public
┣ 📄 nuxt-seo.config.js
```

> - `components/WebLinks.js` untuk [Konfigurasi Frontend](../frontend).
> - Folder `data/` untuk menampilkan identitas dan tautan [sosial media](../update-info#update-biodata-linksdata).
> - Folder `public` adalah tempat dimana kamu menyimpan semua foto.
> - `nuxt-seo.config.js` untuk [mengkonfigurasi SEO](../seo), seperti nama website, pratinjau dan alamat website.
