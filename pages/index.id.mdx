`ADLink` adalah templat link website satu halaman yang dapat disesuaikan.

Tidak perlu menjadi programmer untuk membuat templat ini, cukup mengedit bagian `next-seo.config.js`, `BioData.js` dan `LinkData.js`.

## Fitur

<iconify-icon icon="logos:discord-icon"></iconify-icon> Memanfaatkan komponen [Lanyard](https://github.com/Phineas/lanyard) yang dinamis untuk menampilkan Status Discord.<br /><br/>
<iconify-icon icon="logos:spotify-icon"></iconify-icon> Terinspirasi dari [Spotify Github Profile](https://github.com/kittinan/spotify-github-profile), dikemas menjadi sebuah komponen yang berbeda. Menampilkan lagu yang sedang diputar di Spotify menggunakan [Spotify Status](../frontend#menambah-status-spotify).<br /><br/>
<iconify-icon icon="logos:instagram-icon"></iconify-icon> Jika akun Instagram kamu bersifat publik, konten (contohnya; postingan, reel, panduan) atau profil kamu bisa dibagikan oleh sumber pihak ketiga seperti blog dan artikel menggunakan kode sematan. Kode sematan memungkinkan profil atau konten kamu dibagikan bersama dengan nama pengguna dan tautan ke profil Instagram kamu.<br /><br/>
<iconify-icon icon="logos:bash-icon"></iconify-icon> Daftar website yang dibuat menggunakan [ADLink](https://bit.ly/ADLink24) dan [ADResume](https://bit.ly/AD-Resume), lihat dan silahkan tambahkan website Anda ke [Showcase](/showcase).
