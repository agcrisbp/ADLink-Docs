### Enable/Disable Social Media
- Find `type: social` and change `on:true|false`.

| Title           | on (default)    | 
| :------------ | ----------: |
| `Truth Social`       | `true`          | 
| `Twitter`       | `true`          | 
| `Instagram`     | `false`          | 
| `Medium`        | `false`          | 
| `Github`        | `true`          | 
| `Youtube`       | `true`          | 
| `Linkedin`      | `true`          | 
| `Email`         | `true`          |
| `Tiktok`    | `false`          | 
| `Post.news`      | `false`          | 

<Callout type="info">
- Setting `on: true` show the social icon. 
- The social media icons are arranged in a single row at the top of the page below description. If you want to use as list, change type to `type: 'other'`.
</Callout>

### Add New Social Media Link

- Create a new block by copying this:
```js filename="data/LinksData.js" copy
{
   title: 'Social Name',
   url: 'https://example.com/',
   type: 'social',
   icon: '/icon.png',
   on: true
}
```

<Callout type="info">
Update all info and make sure to add a `icon.png` file in [public](../directory-structure) folder.
</Callout>
