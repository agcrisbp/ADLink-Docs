<Callout type="info">
Kamu hanya perlu mengubah file `next-seo.config.js`.
</Callout>

```js filename="next-seo.config.js" showLineNumbers
export default {
    openGraph: {
        type: 'website',
        locale: 'en_IE',
        title: 'Your Name',
        url: 'Your Website URL with https',
        description: 'About you.',
        locale: 'en_EN',
        keywords: 'all, keywords, goes, here',
        images: [
            {
                width: 1200,
                height: 630,
                url: '/preview.png',
            },
        ],
        site_name: 'Your Website Name',
    },
    twitter: {
        handle: '@agcrisbp',
        site: 'Your Website URL without https',
        cardType: 'summary_large_image',
    },
};
```
